Formation en Base de données

La formation en Base de données vous apprend à concevoir, créer, gérer et sécuriser les données d'une entreprise à l'aide de systèmes de gestion de bases de données modernes.

Programme de la formation

  • Introduction aux bases de données
  • Modélisation des données (MCD, MLD)
  • Langage SQL
  • MySQL
  • phpMyAdmin
  • Création et gestion des tables
  • Requêtes SQL
  • Sécurité et sauvegarde des données

Durée

6 mois

Débouchés

  • Gestionnaire de bases de données
  • Assistant administrateur de bases de données
  • Technicien informatique
  • Développeur Back-end débutant
